# Cinema

## Описание
Проект находится на этапе реализации. На текущий момент сделаны экраны onboarding, главная страница с категориями фильмов и детальная старница фильма.

## Стэк технологий
- MVVM
- Retrofit, okhhtp
- Dagger
- Coroutines
- ViewBinding
- Paging
